# Security Policy

## Supported Versions
This project is educational and does not collect sensitive user data. 
All versions are considered safe for use.

| Version | Supported |
| ------- | --------- |
| All     | :white_check_mark: |

## Reporting a Vulnerability
While this project is low-risk, if you discover a security issue, please report it responsibly by sending an email to:

Awesome-Calculators@proton.me

We will review reports promptly and take necessary action if needed. 
Note: No sensitive user data is collected by this project.

## Disclaimer
This project is maintained as a safe, educational set of calculators and converters. 
Although we aim for maximum safety and reliability, the maintainers cannot be held responsible for any issues arising from its use. 
Use it for learning, experimenting, and having fun, and you should be fine! ^^
